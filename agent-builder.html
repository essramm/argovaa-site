Quick connect — AWS S3 example
Python 
import argovaa

# Init agent with your API key
agent = argovaa.Agent(
  api_key="arg_live_xxxx",
  name="Cost analyst",
  model="claude-3-5-sonnet"
)

# Point to your data — one line per source
agent.connect("s3://my-bucket/cost-data/")
agent.connect("postgresql://prod-db/analytics")
agent.connect("https://api.myservice.com/metrics")

# Deploy — get a REST endpoint instantly
endpoint = agent.deploy()
# https://agents.argovaa.com/v1/agent_xxxx

# Ask a question
result = agent.ask("What drove our cost spike on March 14?")
print(result.answer)

Full integration guide
INTEGRATE YOUR
SERVICE IN 4 STEPS

From sign-up to production agent in under 10 minutes.

01
Sign in & get API key
Create your Argovaa account. Your API key is in Settings → Agent keys. Paste it into the SDK init call.
# Install the SDK
pip install argovaa-sdk

# Your key is in Settings
api_key = "arg_live_xxxx"
02
Create your agent
Choose a template or define your own system prompt. Set the model, tone, and capabilities for your specific use case.
agent = argovaa.Agent(
  api_key=api_key,
  name="My analyst",
  system_prompt="You are...",
  model="claude-3-5-sonnet"
)
03
Connect your data
Pass connection strings for each data source. Argovaa handles authentication, schema discovery, and query generation automatically.
agent.connect("s3://bucket/data/")
agent.connect("postgresql://db/prod")
agent.connect("https://api.co/v1",
  auth={"token": MY_TOKEN})
04
Deploy & go live
Call deploy() to get a live endpoint. Embed the widget in your product or call the API. Monitor usage and costs in the Argovaa dashboard.
ep = agent.deploy()
# → agents.argovaa.com/v1/agent_x

# Or embed a chat widget
<script data-agent="agent_x"
  src="cdn.argovaa.com/widget.js"
></script>
